# 🦜 لینگو - انگلیسی را با شادی یاد بگیرید!

<div align="center">

🇮🇷 [فارسی](./README.fa.md) | 🇺🇸 [English](./README.md)

</div>

یک برنامه وب تعاملی طراحی شده برای فارسی‌زبانان که به آن‌ها کمک می‌کند تا اصطلاحات انگلیسی را یاد بگیرند و دانش خود را از طریق مسابقات جذاب محک بزنند. ساخته شده با فناوری‌های مدرن وب برای تجربه‌ای روان و جذاب.

## 🎯 هدف

لینگو به طور خاص برای **فارسی‌زبانان** طراحی شده است تا انگلیسی را از طریق دو روش قدرتمند یاد بگیرند:

### 1. 📚 دیکشنری اصطلاحات
- یادگیری بیش از ۱۰۰ اصطلاح انگلیسی با ترجمه فارسی
- یادگیری از طریق جملات مثال واقعی
- درک کاربرد اصطلاحات با معنی مثال‌ها

### 2. 🎯 مسابقه چهارگزینه‌ای
- محک زدن دانش انگلیسی با سوالات چهارگزینه‌ای
- یادگیری نکات جالب درباره زبان و فرهنگ انگلیسی
- پیگیری امتیاز و پیشرفت

## ✨ امکانات

### 📚 بخش اصطلاحات

#### مجموعه کامل اصطلاحات
- مرور اصطلاحات به ترتیب حروف الفبا یا تصادفی
- هر اصطلاح شامل:
  - عبارت انگلیسی
  - تلفظ تلفنتیک
  - ترجمه فارسی
  - جمله مثال (با هایلایت اصطلاح)
  - معنی مثال به فارسی

#### 🔍 جستجو و فیلتر
- جستجوی اصطلاحات به انگلیسی یا فارسی
- فیلتر کردن نتایج در لحظه
- نمایش تعداد اصطلاحات منطبق

#### 🎲 تصادفی هوشمند
- حالت «تصادفی» با انیمیشن تاس
- بهم زدن اصطلاحات برای تمرین متنوع
- انیمیشن‌های روان هنگام بهم زدن

#### 🧭 ناوبری بصری
- دکمه‌های بعدی/قبلی برای یادگیری پله‌پله
- کلیک روی هر اصطلاح از لیست
- رابط کاربری مناسب صفحه‌کلید (پشتیبانی از RTL برای فارسی)

### 🎯 بخش مسابقه

#### نمای شبکه سوالات
- نمایش همه سوالات به صورت کارت‌های قابل کلیک
- پیگیری پیشرفت بصری
- انتخاب سوال تصادفی

#### تجربه مسابقه تعاملی
- سوالات چهارگزینه‌ای درباره زبان انگلیسی
- بازخورد فوری به پاسخ‌ها
- نکات جالب بعد از هر سوال
- پیگیری امتیاز در طول مسابقه

#### ناوبری روان
- جریان سوالات پشت‌سرهم
- دکمه‌های سوال قبلی/بعدی
- بازگشت به نمای شبکه پس از اتمام

### 🎨 رابط کاربری زیبا
- پس‌زمینه گرادیانت واکنش‌گرا
- انیمیشن‌ها و انتقال‌های روان
- پالت رنگی مناسب حالت روشن/تاریک
- افکت هاور و بازخورد بصری
- پشتیبانی کامل از RTL برای زبان فارسی
- طراحی واکنش‌گرا برای موبایل

## 🎨 انیمیشن‌های کلیدی CSS

برنامه شامل انیمیشن‌های سفارشی زیر است:
- `bounce` - برای آیکون‌های کارت‌های صفحه اصلی
- `diceRoll` - برای دکمه تصادفی در اصطلاحات
- `slideIn` - برای آیتم‌های لیست اصطلاحات هنگام بهم زدن
- `fadeIn/fadeOut` - برای انتقال روان پنل جزئیات
- `pulse` - برای نشانگرهای بارگذاری
- `scale` - برای بازخورد تعاملی دکمه‌ها

## 🌍 محلی‌سازی

رابط کاربری به طور کامل برای پشتیبانی از زبان فارسی بهینه‌سازی RTL (راست به چپ) شده است. برنامه از موارد زیر استفاده می‌کند:
- چیدمان RTL (`dir="rtl"`)
- ترجمه‌های فارسی برای تمام محتوای آموزشی
- قابلیت جستجوی فارسی
- متن فارسی رابط کاربری

## 🤝 مشارکت

این یک پروژه **متن‌باز** است! برای مشارکت:

### اضافه کردن اصطلاحات جدید
1. مخزن را فورک کنید
2. اصطلاحات خود را به `/public/community/idioms.json` اضافه کنید
3. ساختار JSON را حفظ کنید
4. درخواست pull ارسال کنید

### اضافه کردن سوالات جدید مسابقه
1. سوالات را به `/public/community/english-trivia.json` اضافه کنید
2. اطمینان حاصل کنید هر سوال دارای:
   - متن سوال واضح و دقیق
   - ۴ گزینه پاسخ (یکی صحیح)
   - یک نکته جالب آموزشی
3. درخواست pull ارسال کنید

### راهنمای مشارکت
- اطمینان از ترجمه‌های دقیق فارسی
- ارائه جملات مثال واقعی برای اصطلاحات
- گنجاندن تلفظ تلفنتیک (IPA توصیه می‌شود)
- تأیید معانی مثال به فارسی
- جذاب و آموزشی بودن سوالات مسابقه
- مختصر و مفید بودن نکات جالب
